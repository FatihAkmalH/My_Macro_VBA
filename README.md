# Excel VBA Macro Repository (Internal)

Repositori ini berisi kumpulan **macro VBA Microsoft Excel** yang dikembangkan sebagai bagian dari pekerjaan internal. Macro digunakan untuk mendukung proses **pengolahan data, otomasi laporan, standarisasi format, dan efisiensi workflow** di lingkungan kerja.

Seluruh isi repositori ini diperuntukkan **khusus penggunaan internal**.

---

## 🎯 Tujuan
- Mendokumentasikan macro VBA yang digunakan dalam proses kerja
- Menjaga konsistensi dan standarisasi pengolahan laporan
- Mempermudah pemeliharaan dan pengembangan macro di masa mendatang
- Mengurangi pekerjaan manual dan risiko human error

---

## 📁 Ruang Lingkup Macro
Macro dalam repositori ini mencakup, namun tidak terbatas pada:
- Formatting dan standarisasi laporan
- Pengolahan dan pembersihan data
- Otomatisasi copy, merge, dan distribusi data
- Pembuatan dan penyimpanan file laporan otomatis
- Utilitas pendukung (pengaturan sheet, kolom, dan layout)

---

## 📂 Struktur Repositori
Contoh struktur file:
```
/macro
  ├─ format_laporan.bas
  ├─ move_copy.bas
  ├─ export_data.bas
  └─ utility.bas
```

atau file Excel macro-enabled:
```
/excel
  ├─ template_laporan.xlsm
  └─ automation_report.xlsm
```

---

## ⚙️ Cara Penggunaan
1. Buka file Excel macro-enabled (`.xlsm`)
2. Aktifkan macro saat diminta
3. Jalankan macro melalui:
   - Menu `Developer → Macros`
   - Shortcut `ALT + F8`
   - Tombol pada worksheet (jika tersedia)
4. Pastikan struktur data sesuai dengan ketentuan macro sebelum dijalankan

---

## ⚠️ Ketentuan & Catatan Penting
- Macro dikembangkan berdasarkan struktur data tertentu
- Perubahan nama sheet, kolom, atau posisi data dapat memengaruhi hasil
- Disarankan melakukan backup file sebelum menjalankan macro
- Modifikasi macro hanya dilakukan oleh personel yang memahami VBA
